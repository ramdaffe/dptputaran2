# DPT Putaran 2

Repository for code and map visualizing the difference between number of registered voters in second round compared to first round of DKI Jakarta 2017 election. 

![Difference in Registered Voters](https://github.com/ramdaffe/dptputaran2/blob/master/kelurahan-map.jpg?raw=true)

Interactive map version can be viewed [here.](https://ramdayz.carto.com/viz/42f1a76c-21e4-11e7-a274-0e3a376473ab/public_map)

Summary file for kelurahan level data: [diff-kelurahan.csv](https://github.com/ramdaffe/dptputaran2/blob/master/diff-kelurahan.csv)

For kota & kecamatan level data: [diff-kabkec.csv](https://github.com/ramdaffe/dptputaran2/blob/master/diff-kabkec.csv)

Jupyter Notebook to extract data from KPU website: [dpt-extract.ipynb](https://github.com/ramdaffe/dptputaran2/blob/master/dpt-extract.ipynb)
